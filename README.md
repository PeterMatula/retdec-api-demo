# retdec-api-demo

Small demo that shows how to use RetDec API Poc in other projects.

## Requirements

* Works only on Linux.
* Everything that [RetDec](https://github.com/avast/retdec#requirements) requires.

## Build

```
mkdir build
cd build
cmake ..
make -j<N>
```

## Use

```
build/demo/demo -i tests/<file>
```
